Project Period

JumpJumpSlime: 2018.06 ~ 2019.03

USE SKILLS

개발 언어 : JavaScript(nodejs)

사용프레임워크 : express,mysql,async,express-session,express-mysql-session

개발 환경 : eclipse

사용 DB : My_sql

server : nginx,pm2

    개발 내용
  • 게임 정보를 DB에 저장하기 위해 AWS에 서버 구축
  • 웹서버는 nginx 설치하였고 routing 작업으로 nodejs port로 접근 하도록 처리
  • pm2 module를 이용하여 nodejs서버를 지속적으로 띄우도록 하며 Load Balancing 처리하여 가상의 was 2개 서버 운영
  • DB처리는 async module를 이용하여 기본적으로 Transaction 처리가 이루어 지도록 코딩
  • express-mysql-session를 이용하여 session정보를 DB에 담도록 처리
